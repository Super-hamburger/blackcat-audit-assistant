# BlackCat Audit Assistant 4.4.4

发布日期：2026-07-18

## 发布说明

- 文件粘贴按 Windows 安全文件名分别输出 SKU×1、SKU×N和多SKU、完整成品表三类文件。
- 面单打印支持全部客户一次打印或按客户逐个打印；投函和宅急便可合并或分开处理，并按货架顺序输出。
- 扫码验单的异常改为非模态强红色视觉反馈和提示音，连续扫码不会被弹窗阻塞。
- 扫码工作台将整体已匹配/总数/百分比进度置于扫码输入下方，并以高对比度卡片突出当前出库单状态；明细、最近记录和未匹配源数据导出保留在次级区域。
