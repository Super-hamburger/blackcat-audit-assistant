# Help

## 第一次使用

1. 文件粘贴：选择 Excel，选择输出位置，生成上传表。
2. 面单压缩：选择一个或多个 PDF，选择输出目录，开始处理。
3. 如果文件很多，建议先放到同一个文件夹，再多选导入。

## 安装新版本

1. 在“设置 -> 检查更新”确认新版本号、下载地址和 SHA256。
2. 选择立即更新后，软件会下载并校验安装包。
3. 确认安装后，软件会退出，由独立脚本覆盖安装并重新启动。
4. 如果一键更新失败，再下载 GitHub Release 里的新版安装包 ZIP。
5. 解压 ZIP，运行 `Install_BlackCatAuditAssistant.bat` 覆盖安装。
6. 重新打开软件，确认标题栏版本号已更新。

## 发布规则

- 默认先做本机测试版。
- 只有明确确认“发布新版本”后，才更新 GitHub Release 和 GitHub Pages 的远程清单。
- 一键更新依赖远程清单里的 `download_url` 和 `package_sha256`，发布时必须填写真实值。
